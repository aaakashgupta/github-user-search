# 🚀 GitHub User Search

A modern React application that allows users to search GitHub profiles using the GitHub REST API. Users can search any GitHub username and view profile details like avatar, name, followers, following, and public repositories.

## ✨ Features

- 🔍 Search GitHub users by username
- 👤 Display user profile information
- 🖼️ Show profile avatar
- 👥 Followers & Following count
- 📦 Public repositories count
- ⚡ Fast and responsive interface
- 📱 Mobile-friendly design

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Flowbite React
- React Icons
- GitHub REST API

## 🚀 Installation

```bash

cd github-user-search

npm install

npm run dev
```

## 🌐 GitHub API

This project uses the GitHub REST API.

Example:

```text
https://api.github.com/users/octocat
```
